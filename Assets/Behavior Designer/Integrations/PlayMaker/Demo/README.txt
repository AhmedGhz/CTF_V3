This demo scene uses Playmaker Global Variables that must first be imported. You can import them with the following steps:

If you already have global variables:
- Open your Playmaker Global Variables file located within PlayMaker/Resources/PlayMakerGlobals.asset
- Click the Import Globals button
- Select the package located at Behavior Designer/Integrations/PlayMaker/Demo/PlayMakerGlobalsExport.unitypackage

If you do not have any global variables:
- Import the global variables package at Behavior Designer/Integrations/PlayMaker/Demo/PlayMakerGlobalsNew.unitypackage
